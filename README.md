# 🏆 Badminton Elite

A high-performance, immersive booking and academy platform for badminton enthusiasts. Engineered with a "God-Level" design aesthetic, featuring fluid animations and a premium interface.

## ✨ Key Features

- **⚡ Kinetic Ticker**: Real-time updates with dynamic CSS-based scrolling.
- **📅 Advanced Booking System**: Seamless court reservation experience with interactive UI.
- **🎭 Framer Motion Animations**: Smooth transitions and high-fidelity micro-interactions.
- **📱 Responsive Design**: Fully optimized for all device sizes, from mobile to ultra-wide displays.
- **🏸 Academy Programs**: Detailed insights into training tiers and elite mastery sessions.

## 🛠 Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS with modern custom properties
- **Language**: TypeScript

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI components (Hero, BookingSystem, KineticTicker, etc.).
- `src/utils`: Helper functions and services.
- `public`: Static assets.
